Detailed Description

The StyleSample class provides to view a square color.

Inherits QWidget.

By default square color is set to TPixel32(235,235,235,255), you can set other color using setColor(); you can also define current color with a style TColorStyle, getStyle(), using setStyle(). You can pass to constructor square size.

StyleSample permit to manage click event, it's possile to enable this feature setting enableClick(bool on) to true. If it is enable when click in square class emit the signal clicked(const TColorStyle &style).
