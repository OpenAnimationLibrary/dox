Detailed Description

The TStageObjectSpline class define stage object motion path.

Inherits TSmartObject and TPersist.
The motion path is defined by spline build as TStroke getStroke(), and node position getDagNodePos(); spline can be set using setStroke() while node position can be set using setDagNodePos().
It's possible to clone spline using clone().
