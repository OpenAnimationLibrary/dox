Detailed Description

The class TStageObjectId permits the specialization of a generic stage object.

The possible type of a stage object are: table TableId, camera CameraId, pegbar PegbarId, column ColumnId and spline SplineId; the class allows to know object type: isCamera(), isTable(), isPegbar(), isColumn() and isSpline().
For each specialization you give to each stage object a sequential number. The class provides methods to get sequential number getIndex() and objectId name toString(). It's possible to compare different stage object using defined operator, operator==(), operator!=(), operator<(), operator<=(), operator>() and operator>=().
