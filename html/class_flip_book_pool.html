Detailed Description

The FlipBookPool class is used to store used flipbook viewers.

Flipbooks are generally intended as temporary but friendly floating widgets, that gets displayed when a rendered scene or image needs to be shown. Since a user may require that the geometry of a flipbook is to be remembered between rendering tasks - perhaps even between different Toonz sessions - flipbooks are always stored for later use in a FlipBookPool class.

This class implements the basical features to pop a flipbook from the pool or push a used one; plus, it provides the save and load functions for persistent storage between Toonz sessions.
