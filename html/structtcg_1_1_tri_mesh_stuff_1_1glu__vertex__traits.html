Detailed Description

template<typename vertex_type>
struct tcg::TriMeshStuff::glu_vertex_traits< vertex_type >

Traits class providing useful accessor to vertex data in a form compatible with the GLU tessellator.

Vertex types usable with the GLU tessellator must have:

  • A vertex position member, in the form of a double[3] array containing its (x,y,z) coordinates.
  • An index member to identify the vertex.
