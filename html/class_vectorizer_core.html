Detailed Description

Contains specific vectorization methods and deals with partial progress notifications (using Qt signals).

VectorizerCore class is the lowest layer of a vectorization process, it provides vectorization of a single input raster image by calling the vectorize method.

It can also deal notifications about its progress status, and is receptive to user cancels.

VectorizerPopup, Vectorizer, VectorizerConfiguration classes.
