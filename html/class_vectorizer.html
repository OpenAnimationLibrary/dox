Detailed Description

The thread encapsulating a core vectorization process.

The Vectorizer class serves as a medium between Xsheets and the low-level vectorization methods: each frame of the input level is processed by VectorizerCore methods and the output is then stored on free xsheet cells. 

      Signals are also excanghed through this class, ensuring basic

communications between the invoker (typically VectorizerPopup) and the VectorizerCore class.

Since vectorization is typically a complex and resource-consuming task, Vectorizer also inherits a QThread in order to avoid system freezes.

See also
The VectorizerPopup, VectorizerConfiguration and VectorizerCore classes.
