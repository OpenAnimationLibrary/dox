Detailed Description

The TXshLevel class is the base class of level managers in xsheet.

Inherits TSmartObject and TPersist.
This is an abstract base class inherited by the concrete classes TXshChildLevel, TXshSimpleLevel and TXshZeraryFxLevel.

The class contains all features about a generic level and gives all methods to access to these information.

It's possible ot know level name, getName(), or identify level types getType(), defined in txshleveltypes.h, and change it, setType(). The class provides function to know scene to which level belongs, getScene(), and function to change scene setScene().
