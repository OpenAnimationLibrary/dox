Detailed Description

The DoubleValueField class provides to view an object to manage a generic double parameter, with two cipher after decimal point. 

            Inherits \b QWidget.

            The class is composed of an horizontal layout which contains two

object,

  • a vertical layout with a text field PrimaryDoubleLineEdit and a roller
  • an horizontal slider QSlider. This objects are connected, so if you change one the other automatically change. It's possible to set current value getValue(), minimum and max value, getRange(), using setValue(), setRange() or setValues(); by default value = 0, minimum value = -100.0 and maximum value = 100.0.
    Maximum height object is fixed to DVGui::WidgetHeight, while width depend on parent width.

To know when double parameter value change class provides a signal, valueChanged(); class emit signal when slider value change or when editing text field is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT: onSliderChanged(int value), onEditingFinished().
