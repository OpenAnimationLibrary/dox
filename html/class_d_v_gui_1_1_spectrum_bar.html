Detailed Description

The SpectrumBar class provides a bar to display a spectrum. 

         Inherits \b QWidget.

         The object allows to manage a spectrum, permit you to insert,

move and remove key.

You can set spectrum default color in constructor, or set color of current key, getCurrentColor(), using setCurrentColor(). It's possible to change current key, getCurrentKeyIndex(), through its index, setCurrentKeyIndex(). You can also set current key position, getCurrentPos(), using setCurrentPos() or add a new key in a specific position, addKeyAt().
