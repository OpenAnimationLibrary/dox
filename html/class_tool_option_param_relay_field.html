Detailed Description

The ToolOptionMeasuredDoubleField class implements toolbar controls for double properties that need to be displayed with a measure.

This option control is useful to display function editor curves in the toolbar; in particular, it deals with the following tasks:

  • Setting the preference-based keyframe interpolation type
  • Editing with global keyframes (ie affecting multiple parameters other than the edited one)
  • Undo/Redo of user interactions.
