Detailed Description

template<typename RanIt>
class tcg::polyline_ops::StandardDeviationEvaluator< RanIt >

Calculates the (weighted) standard deviation of a polyline's sub-paths with respect to the segment connecting the endpoints.

Warning
For efficiency reasons, the returned value is the actual standard deviation, times the endpoints-segment length.
