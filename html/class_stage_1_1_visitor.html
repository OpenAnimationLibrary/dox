Detailed Description

Stage::Visitor is the abstract class model related to scene visiting <\I>.

Iterating a scene to perform some task on its content is a typical task in Toonz. Scene visiting is here intended as an implementation of the visitor-visitable <\I> pattern applied to Toonz scenes, where the visitable objects are of type Stage::Player, and visitors are reimplemented from the Stage::Visitor class.

The Stage::Player class is the rough equivalent of a level at a specific scene frame (the term Player being more legacy rather than having a definite sense I guess), which in particular can be seen as an image occurring in scene compositing. Stage::Player actually have a lot more insights about onion skinning, image placement, and other stuff - be sure to check it thorough.

When visiting a scene, the Players are supplied to Stage::Visitor instances sorted in compositing order - so e.g. the first Player is intended below the last one.
