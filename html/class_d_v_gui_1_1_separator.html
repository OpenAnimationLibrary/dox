Detailed Description

The Separator class provides a separator. 

        Inherits \b QWidget.

        The separator can be text and line or only line. If QString \b

name, passed to constructor, is not empty, separator is composed by text name and line; else separator is a line, this line width is DV dialog width, clearly taking care DV dialog margin. The separator can be horizontal (by default) or vertical, isVertical(), you can set it using function setOrientation().

To add a separator to DV dialog Dialog you must create a new Separator and recall Dialog::addWidget(), or recall Dialog::addSeparator(). 

         \b Example: in a DV dialog \b Dialog
         \code
                 Separator* exampleNameAndLine = new

Separator(QString("Example Name")); addWidget(exampleNameAndLine); Separator* exampleLine = new Separator(""); addWidget(exampleLine); or:

addSeparator(QString("Example Name"));
addSeparator();
