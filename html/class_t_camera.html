Detailed Description

The TCamera class provides a camera and allows its management.

A camera is specified by size, getSize() and resolution, getRes(). It can be changed using the setSize(), setRes() functions.tcamera

The class moreover gives methods to know camera Dpi getDpi(), camera aspect ratio getAspectRatio() and stage rect getStageRect().
