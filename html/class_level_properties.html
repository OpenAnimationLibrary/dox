◆ setSubsampling()

void LevelProperties::setSubsampling ( int  s)

Sets the level subsampling.

The level subsampling is the integer value (strictly greater than 1) denoting the fraction of pixels to be displayed for its images while in camera stand mode. For example, a value of 2 will halve the original image resolution.

Note that only the crudest algorithm will be applied for resampling, as this feature is explicitly intended to improve visualization speed as much as possible.
