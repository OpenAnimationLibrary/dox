Detailed Description

The SceneSoundtrack class provides a container for scene sound track resource and allows its management.

Inherits SceneResource.
The class, more than SceneResources, contains a pointer to a sound column TXshSoundColumn, two path for old save path, one decoded and one no. The class allows to save sound track save() and to update or to restore old path updatePath(), rollbackPath().
