Detailed Description

Dockable widgets are widget containers handled by DockLayout class.

DockLayouts accept only dock widgets of this class; so if you want to place a given widget under this kind of layout, a DockWidget shell for it must be allocated first. The following class implements base dock widgets, with native floating decorations and no docked title bar. It is encouraged to reimplement all the essential functions for custom aspect and behaviour.

See also
DockLayout and DockPlaceholder classes.
