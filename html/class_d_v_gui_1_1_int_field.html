Detailed Description

The IntField class provides to view an object to manage an integer parameter.

Inherits QWidget.

The class is composed of an horizontal layout which contains two object,

  • a vertical layout with a text field IntLineEdit and a roller
  • an horizontal slider QSlider. This objects are connected, so if you change one the other automatically change. You can set current value getValue(), minimum and max value using setValue(), setRange() or setValues(); by default is value = 0, minimum value = 0 and maximum value = 100.
    Maximum height object is fixed to DVGui::WidgetHeight, width depend on parent width.

To know when integer parameter value change class provides a signal, valueChanged(); class emit signal when slider value change or when editing text field is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT: onSliderChanged(int value), onEditingFinished().
