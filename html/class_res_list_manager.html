Detailed Description

The ResListManager is a singleton class which manages presets of camera resolutions.

This class allows a lightweight management of the resolution presets used in camera settings. Specific tasks include resolution fetching and the add/remove functions. Adding or removing presets causes the listChanged() signal to be emitted.

Note
There actually are two distinct instances of this class, one managing standard cameras and the other for cleanup cameras. Presets are stored in two distinct files as well.
