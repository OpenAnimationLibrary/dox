Detailed Description

This class is the interface for manipulation algorithms.

StrokeDeformation is a wrapper for for StrokeDeformationImpl, its function is to verify state and parameteres before to call the implementation methods of deformation.

There is an internal status to verify that actions:

  • CREATED on constructor
  • ACTIVE on active
  • UPDATING on updating
  • DEACTIVE on deactive

If some error of state occurs reset method is called. ContextStatus contains information about deformation parameter, and it occurs to reduce the number of parameters required.
