Detailed Description

template<typename T, typename B = empty_type>
class tcg::safe_bool< T, B >

The tcg::safe_bool template class is a simple implementation of the classic Safe Bool Idiom, meant to be used inside tcg.

For a better implementation, you should consider looking into Boost.Spirit.Safe_Bool.
