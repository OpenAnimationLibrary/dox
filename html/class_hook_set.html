Hook * HookSet::addHook ( )

Adds a hook to the set.

A Hook can be added replacing a null pointer in the Hook container with a new Hook, getting an empty hook or adding a new Hook at the end of the container.

It is possible to istantiate at max HookSet::maxHooksCount hooks - beyond which this function will return 0.
