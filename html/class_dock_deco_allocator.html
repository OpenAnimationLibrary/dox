Detailed Description

DockDecoAllocator class handles the allocation of decoration elements used by our dock manager.

In order to implement custom appearances for the docking system, it is possible to customize both DockSeparator and DockPlaceholder classes. Since allocation of such objects is handled internally by the docking system, it is necessary to reimplement allocator functions whenever decoration classes change. In order to assign a DockDecoAllocator to a DockLayout or DockWidget, the respective 'setDecoAllocator' methods are provided.

See also
DockLayout, DockWidget, DockSeparator and DockPlaceholder classes.
