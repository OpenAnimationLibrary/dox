Detailed Description

The StudioPaletteViewer class provides an object to view and manage studio palettes.

The StudioPaletteViewer class provides an object to view and manage studio palettes.

Inherits QFrame. This object is composed of a splitter QSplitter that contain a vertical layout and a PaletteViewer. Vertical layout contain a StudioPaletteTreeViewer and a toolbar, this object allows to manage the palettes in studio palette folders. PaletteViewer is set to fixed view type: PaletteViewerGUI::STUDIO_PALETTE allows to show and modify current studio palette selected in tree.
