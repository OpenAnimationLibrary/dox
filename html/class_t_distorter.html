Detailed Description

The TDistorter class provides the base template for implementing custom image distorters that work with the distort() method.

virtual int TDistorter::invMap ( const TPointD &  p,
TPointD *  results 
) const
pure virtual

Returns the pre-images of distorted point p. The results are overed on the output image as they appear in the returned vector, so the user must be sure to sort them accordingly. NOTE: The results array is already allocated with the number of entries specified by the maxInvCount() function in order to avoid allocation at this level; however, the number of effective inverses must be specified as a return value.

Implemented in Warper< T >, BilinearDistorterBase, BilinearDistorter, and PerspectiveDistorter.
