Detailed Description

The ShortcutZoomer abstract base class is used by viewer widget to access shortcut-related commands.

This class is a wrapper for shortcuts established by the CommandManager interface.

Subclass it defining the required view commands, then implement a keyPressEvent() event handler in the viewer widget you want:

void MyViewer::keyPressEvent(QKeyEvent* ke)
{
if(ViewerZoomer(this).exec(event))
return;
return MyViewerBase::keyPressEvent(ke);
}
