Detailed Description

PaletteIconWidget class provides a widget to show a palette icon. 

 \brief    Special placeholder toolbar icon for \a starting a palette

move through drag & drop.

This widget is currently employed as a mere mouse event filter to propagate drag & drop starts to a PaletteViewer ancestor in the widgets hierarchy.
