Detailed Description

The PlasticDeformer class implements an interactive mesh deformer.

Warning
Objects of this class expect that the mesh and rigidities supplied on construction remain constant throughout the deformer's lifetime. Deforming a changed mesh is not supported and will typically result in a crash. Deforming a mesh whose vertex rigidities have been deleted will result in a crash. Altering the rigidities results in undefined deformations until the deformer is recompiled against them.
