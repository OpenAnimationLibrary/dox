Detailed Description

The class responsible for icons management in Toonz.

It's a singleton class - in particular, rendered icons are stored in a shared map container for fast retrieval upon repeated icon requests. 

      IconGenerator provides methods to submit icon requests, and to

invalidate or remove icons from the internal database. In order to keep outer entities informed of the icon generation status, an iconGenerated() signal is emitted once an icon has been generated.
