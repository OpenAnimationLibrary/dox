Detailed Description

The FrameNavigator is a simple toolbar widget showing a numerical representation of an integer timeline.

A FrameNavigator is a TFrameHandle viewer composed of a line edit widget showing the current frame number, and a pair of directional arrow buttons that move to adjacent frames.

Remarks
The text field actually visualizes the internal integer m_frame +1 (e.g. when m_frame is 0 then 1 is visualized).
