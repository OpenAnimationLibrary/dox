Detailed Description

ResourceImportStrategy allows to control the resource import behaviour during load operation. It can be statically defined (i.e. DONT_IMPORT). Subclass it to provide dialog boxes. All the actual file copies are recorded (ImportLog).

See also
ResourceImportMap, ResourceImportDialog (defined in toonz)
