Detailed Description

The PageViewer class provides an object to view and manage palettes page. 

         Inherits \b QFrame and \b TSelection::View.

         This object allows to show and manage a single palette page. An

enum type PaletteViewerGUI::PageViewer::ViewMode, getViewMode(), specify visualization mode, it is possible set it using setViewMode(). Class manage all mouse event, move, press, drag and drop.

◆ ViewMode

enum PaletteViewerGUI::PageViewer::ViewMode

Possible view modes for a Palette Viewer.

Useful to mark different page view moed:

  • SmallChips: displays styles as small squares; the name of the style is displayed as tooltip.
  • SmallChipsWithName: displays styles as small squares with the name inside.
  • LargeChips: displays the styles on top of the name of the style.
  • List: List displays style thumbnails next to their names in a list; if the palette styles are referring to a studio palette, its path will be displayed along with the style name.
