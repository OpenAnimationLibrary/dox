Detailed Description

template<typename Pix, typename Val>
class TRop::borders::pixel_selector< Pix, Val >

This class is just a stub for pixel selector classes.

Pixel Selectors are objects used in raster borders recognition to interpret a raster's pixels and direct edge iterators.

The main purpose of this class is that of distinguishing pixels from their values <\I>. Values are the actual entities recognized in the borders extraction process - adjacent pixels that are cast to the same value will be included in the same border.

For example, if we want to extract the borders of a fullcolor image that separate bright areas from dark ones, we can just have a pixel selector cast each pixel to the integer value 0 if the pixel's value is below the brightness threshold, and 1 if above.

Also, pixel selectors tell whether pixels are to be intended as completely transparent (virtual pixels outside the raster boundaries are).
