int tcg::TriMesh< V, E, F >::collapseEdge ( int  e)

Collapses the specified edge.

Specifically, this function removes edgeVertex(e, 1) and redirects its edges to edgeVertex(e, 0). One edge per adjacent face (other than e) will be removed (which can be thought as 'merged' with the remaining one), and each adjacent face will be removed.

Note
This function removes at most 1 vertex, 3 edges and 2 faces, total.
Warning
The user is respondible for ensuring that every vertex adjacent to both the collapsed edge's endpoints is also adjacent to one of the edge's faces. If not, the output configuration would be ill-formed. This function will assert in this case, and result in undefined behavior.
Returns
The remaining vertex index - specifically, edgeVertex(e, 0).

Swaps the specified edge in the two adjacent faces.

This function selects an edge with two adjacent faces, and swaps its endpoints with their otherFaceVertex().

Remarks
This function is idempotent - swapEdge(swapEdge(e)) has no effect on the mesh (assuming swapEdge(e) is not -1). In particular, indices should remain the same.
Note
In current implementation, the result is the very same input edge index.
Returns
The swapped edge, or -1 if the supplied edge did not have two adjacent faces.
