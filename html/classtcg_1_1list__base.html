Detailed Description

template<typename T>
class tcg::list_base< T >

Base class for list-like containers based on tcg::list's indexed list implementation.

This class implements a barebone container class which manages a collection of nodes stored in an iternal random access container.

Nodes management is limited to the acquisition and release of individual nodes (see the buyNode() and sellNode() functions).

Links between list nodes must be managed externally.

See also
Template class tcg::list for an explanation of the indexed list container concept.
