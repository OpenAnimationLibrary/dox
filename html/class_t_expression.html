Detailed Description

This class manages a generic expression.

An expression is a sequence of characthers and contains:

  • a CalcNode that is a basic element of the expression that must be evaluated;
  • a Grammar that is a sequence of string patterns that define operations on nodes;
  • a Builder that is a sequence or a tree of calcnodes and operations on that;
