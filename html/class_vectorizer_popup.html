Detailed Description

The "Convert-To-Vector Settings" popup window.

Vectorization interface is segmented among 3 main classes: VectorizerPopup, Vectorizer, and VectorizerCore. VectorizerPopup represents the outermost layer, providing a suitable user interface to choose between available vectorization modes and options, and displays a progress dialog to inform about the status of the underlying process.

In order to launch a suitable vectorization process, an Xsheet or scene cast selection must be active; then the show() method should be called to let the user choose the preferred vectorization options. Each parameter set is passed down to a VectorizerConfiguration class, so see there for more detailed explanations. The apply() method can be alternatively called for default options, avoiding user interaction.

During the process, the progress bar dialog is set on modal behaviour, in order to prevent the user from modifying or removing the frames to be vectorized.

See also
The VectorizerConfiguration, Vectorizer and VectorizerCore classes.
