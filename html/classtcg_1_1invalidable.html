Detailed Description

template<typename Obj, typename Func = tcg::empty_type>
class tcg::invalidable< Obj, Func >

The invalidable template class is a common case of controlled variable access for objects which can be invalidated, and thus require a validator procedure to be invoked strictly before a user access to the object can be allowed.

An invalidable instance wraps a mutable object and adds a boolean to signal its current validity state.

The initial validity of the object can be specified at construction, and explicitly set to the invalid state through the invalidate() method.

Access to the wrapped object requires either the invocation a validator functor through the various operator() overloads, or that the access is explicitly marked as direct by supplying the tcg::direct_access tag.
