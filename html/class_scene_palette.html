Detailed Description

The ScenePalette class provides a container for scene palette resource and allows its management.

Inherits SceneResource.
The class, more than SceneResources, contains a pointer to a palette level TXshPaletteLevel, two path for old save path, one decoded and one no, two path for old scanned path, one decoded and one no. The class allows to save simple level save() and to update or to restore old path (save and scan) updatePath() rollbackPath().
