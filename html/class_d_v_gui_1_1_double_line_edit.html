Detailed Description

The DoubleLineEdit class provides to view an object to manage a double field. 

         Inherits \b LineEdit, set a double validator \b

QDoubleValidator. 

         You can pass to constructor current value, getValue(), minimum

and max value field, getRange(), and number of decimals, getDecimals() or you can set this value using setValue(), setRange() and setDecimals().
