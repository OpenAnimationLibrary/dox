Detailed Description

This is a private class used to convey the TRenderer::startRendering methods into Qt queued slots. This is necessary since these methods implicitly perform event processing, which could cause trouble in case they are invoked from events which must respect a strict ordering.

For example, suppose that a render must be invoked upon a mousePressEvent, and that such event must have been completely processed before the correspondent mouseReleaseEvent is invoked - calling the startRendering method directly by the mousePressEvent may cause the mouseReleaseEvent to be processed before the former's end.
