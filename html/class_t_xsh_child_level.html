Detailed Description

The TXshChildLevel class provides a child level of xsheet and allows its management.

Inherits TXshLevel.
A child level getChildLevel() is a xsheet reduced to level. The class allows to edit xsheet getXsheet(), change xsheet setXsheet() and change scene xsheet setScene(). It's possibile change level name setName() and know how many frame are in level getFrameCount().

The class provides methods to control child level icon, makeIcon(), getIcon() and invalidateIcon().
