Detailed Description

The DummyLayout class implements a layout whose items' geometry is not enforced at all, ignoring parent size bounds or geometry.

A DummyLayout serves as a simple, inert container of layout items. It does not enforce geometry to items added to it, allowing objects using this layout to appear as a 'viewport' for free-geometry items.

Note
As this layout does not effectively manage its objects' geometry, these must be managed outside.
