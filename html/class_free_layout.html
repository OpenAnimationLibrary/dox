Detailed Description

The FreeLayout class implements a layout whose items can be freely positioned, ignoring parent size bounds or geometry.

A FreeLayout instance can be used to achieve results similar to that of a QScrollArea, with a more straightforward API and without restriction to just one item at a time. Please observe that layouts can be inserted too. 

      Specifically, FreeLayout is a convenience subclass of DummyLayout

whose items' sizes are enforced to their sizeHint() - in order to spare users from having to deal with it externally. This complies with the idea that its items are comfortly free to adapt to their preferred size, as it would be in a scrollable environment.
