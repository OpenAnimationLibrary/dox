Detailed Description

Defines the User Interface Types available for a group of TParam objects.

A User Interface Concept is used to associate parameters to the way they should be interactively represented. For example, a TDoubleParamP may be used to represent multiple objects, such as lengths or angles; a couple of TPointParamP may represent vectors, and so on. Observe that each entry in the list is associated to a specific struct of parameter objects.

The list in the Type enum provides all the interface types currently supported by Toonz.
