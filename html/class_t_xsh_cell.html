Detailed Description

The TXshCell class provides a cell of xsheet, this structure contains a level frame.

TXshCell is composed by a pointer to level TXshLevelP and a TFrameId that identifies right frame in level.

The class provides functions that return the cell element. The getImage() function return image associated to cell. The getSimpleLevel() function return simple level TXshSimpleLevel to which cell belongs. The getFrameId() function return TFrameId frame identification of cell. It's possible to know if cell is empty through isEmpty().
