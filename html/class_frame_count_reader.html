Detailed Description

FrameCountReader is the class responsible for calculation of levels' frame counts in the file browser. Since on many file formats this requires to open the level file and scan each frame (MOV-like), and on some machine configurations such a task can be time consuming, we dedicate a separate thread for it - just like the icon generator does. Calculated frame counts are also stored for quick lookup once they have been calculated the first time.
