Detailed Description

template<typename Obj, typename Func = tcg::empty_type>
class tcg::controlled_access< Obj, Func >

The controlled_access template class encapsulate an object that can be accessed only after a functor has been invoked, enforcing access tracking at compile time.

The accessor functor can be specified by explicit template argument or supplying it during access. In case the accessor type is specified, an instance of the functor will be stored together with the accessed variable.
