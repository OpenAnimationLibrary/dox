Detailed Description

The TXshLevelColumn class provides a column of levels in xsheet and allows its management.

Inherits TXshCellColumn.
The class defines column of levels getLevelColumn(), more than TXshCellColumn has a pointer to TLevelColumnFx getLevelColumnFx() and a string to identify icon. The string is an icon identification helpful to level icon management, getIcon() updateIcon().
