Detailed Description

The SceneLevel class provides a container for scene level resource and allows its management.

Inherits SceneResource.
The class, more than SceneResources, contains a pointer to a simple level TXshSimpleLevel, two path for old save path, one decoded and one no, two path for old scanned path, one decoded and one no. The class allows to save simple level save() and to update or to restore old path (save and scan) updatePath() rollbackPath().
