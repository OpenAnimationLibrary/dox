enum DrawingData::ImageSetType

Define possible level image setting.

  • INSERT Insert images before first selected;
  • OVER_FRAMEID Replace images with the same frameId images in data;
  • OVER_SELECTION Replace selected images with images in data.
32{ INSERT, OVER_FRAMEID, OVER_SELECTION };
