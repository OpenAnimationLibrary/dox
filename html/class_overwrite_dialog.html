Detailed Description

This dialog offers a standard 'overwrite file' resolution Dialog.

Sometimes, the system attempts to write to the file path of an already existing file. This dialog presents a standard interface to let users choose the conflict resolution manually.

There are 3 possible resolution choices:

  • Overwrite the existing file
  • Keep the existing file (aborting the write operation)
  • Rename the file to write adding a suffix

Additionally, the dialog could be \t CANCELED, either by closing it or pressing the "Cancel" button.
