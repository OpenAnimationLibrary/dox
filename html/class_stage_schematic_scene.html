Detailed Description

It is the implementation of the SchematicScene representig the QGraphicsScene for the Stage Schematic. All StageSchematicNode, StageSchematicSplineNode and link are pleced and handled in this scene. The scene maintains two mapping: one for StageSchematicNode and one for StageSchematicSplineNode. These mapping can be used to retrieve nodes and spline.
To regenearte the scene use the updateScene().
To oreder nodes in the scene use the reorderScene().
