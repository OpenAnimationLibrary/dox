Detailed Description

The TRenderPort class acts as a spatially-declared listener to a TRenderer instance. TRenderPorts are the formally supported receivers of TRenderer's activity notifications; they provide some basic pure virtual members that are appropriately invoked by the TRenderer when an associated interesting render event takes place.
