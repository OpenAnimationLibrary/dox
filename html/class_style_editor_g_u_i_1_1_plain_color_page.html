Detailed Description

The PlainColorPage is used to control the color parameter. 

        Inherits \b StyleEditorPage.

        The PlainColorPage is made of a \b SquaredColorWheel and a \b

ColorSlider, a collection of ColorChannelControl, and a number of radio button (to control the ColorWheel behaviour).
