enum FileBrowserPopup::Options

Various options used to customize the popup's behavior.

Enumerator
STANDARD 

Standard options.

CUSTOM_LAYOUT 

Prevents standard layout organization at construction, surrendering it

MULTISELECTION 

to the user. Observe that sub-widgets creation is still enforced.

Enable multiple selection in the browser widget.

WITH_APPLY_BUTTON 

Enable if the filebrowser has an apply button next to the OK button

FOR_SAVING 

The popup is for saving. When file selection is released, previously-clicked file name remains so that users can reuse it for inputting the file name. see FileBrowserPopup::onFilePathsSelected
