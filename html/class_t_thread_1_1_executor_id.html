Detailed Description

Contains all the executor data that need to persist until all tasks added by the executor have been processed. Upon creation of an Executor object, it instantiates one ExecutorId for both storing the Executor's data sensible to the underlying code of the Executor manager, and to identify all tasks added through it - by copying the smart pointer to the id into each added task.

Executor and Runnable class.
