Detailed Description

La classe si occupa della visualizzazione dello scene cast.

E' figlia di QSplitter e contiene un albero che visualizza i folder m_treeView e un widget che consente di visualizzare i file m_sceneCastView. I suoi widget sono settati tramite un modello del tipo SceneCastModel.
