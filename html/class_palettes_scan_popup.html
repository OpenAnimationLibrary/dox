Detailed Description

The PalettesScanPopup class provides a modal dialog to import palette in StudioPalette folder.

Inherits Dialog. This object is composed of three sub-object: a FileField to choose a folder, a label QLabel, generally empty, to show palette folder during them import, and a button bar to start import or cancel command.

The class provides a method, setCurrentFolder(TFilePath path), to set folder wherein import palettes. Protected function allows to manage palette import.
