\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_render_port}{TRender\+Port}} class acts as a spatially-\/declared listener to a \mbox{\hyperlink{class_t_renderer}{TRenderer}} instance. TRender\+Ports are the formally supported receivers of \mbox{\hyperlink{class_t_renderer}{TRenderer}}\textquotesingle{}s activity notifications; they provide some basic pure virtual members that are appropriately invoked by the \mbox{\hyperlink{class_t_renderer}{TRenderer}} when an associated interesting render event takes place.
