\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_movie_renderer}{Movie\+Renderer}} is the high-\/level API class responsible for rendering Toonz scenes into movies. In a more generic view, the term \textquotesingle{}movie\textquotesingle{} represents here a generic sequence of images, which may even be kept in memory rather than written to file.
