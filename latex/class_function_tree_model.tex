\doxysubsection{Detailed Description}
The Function Editor\textquotesingle{}s (tree-\/like) {\itshape model}, as in the {\itshape model-\/view} architecture. 

This class represents the data associated to Toonz\textquotesingle{}s Function Editor panels in a view-\/independent way. The model\textquotesingle{}s purpose is that of representing all {\itshape channels} of the objects in a scene. A {\itshape channel} is here intended as an {\itshape animatable parameter} represented by a {\itshape single real-\/valued function}.

Animatable objects are currently subdivided in two main types\+: {\itshape stage objects} (which consist roughly all the objects represented in the {\itshape stage} schematic view, including cameras, spline curves and pegbars), and {\itshape fxs}. \mbox{\hyperlink{namespace_stage}{Stage}} objects typically feature a uniform channels group structure, whereas each fx type have a different set of parameters (and thus channels). Recently, {\itshape column} objects can sport an additional group of channels, related to Plastic skeleton animations (see Tnz\+Ext library).
