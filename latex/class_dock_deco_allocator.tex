\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_dock_deco_allocator}{Dock\+Deco\+Allocator}} class handles the allocation of decoration elements used by our dock manager.

In order to implement custom appearances for the docking system, it is possible to customize both \mbox{\hyperlink{class_dock_separator}{Dock\+Separator}} and \mbox{\hyperlink{class_dock_placeholder}{Dock\+Placeholder}} classes. Since allocation of such objects is handled internally by the docking system, it is necessary to reimplement allocator functions whenever decoration classes change. In order to assign a \mbox{\hyperlink{class_dock_deco_allocator}{Dock\+Deco\+Allocator}} to a \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}} or \mbox{\hyperlink{class_dock_widget}{Dock\+Widget}}, the respective \textquotesingle{}set\+Deco\+Allocator\textquotesingle{} methods are provided.
