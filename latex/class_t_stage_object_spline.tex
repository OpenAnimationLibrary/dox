\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_stage_object_spline}{TStage\+Object\+Spline}} class define stage object motion path. 

Inherits {\bfseries{\mbox{\hyperlink{class_t_smart_object}{TSmart\+Object}}}} and {\bfseries{\mbox{\hyperlink{class_t_persist}{TPersist}}}}. ~\newline
 The motion path is defined by spline build as {\bfseries{\mbox{\hyperlink{class_t_stroke}{TStroke}}}} \mbox{\hyperlink{class_t_stage_object_spline_a68345d3697e0c20003d9b497d8ccacda}{get\+Stroke()}}, and node position \mbox{\hyperlink{class_t_stage_object_spline_a41676d416f33c550d62aeade360bbf0b}{get\+Dag\+Node\+Pos()}}; spline can be set using \mbox{\hyperlink{class_t_stage_object_spline_a3a08efafca4cc8e0ddf06ffa13898340}{set\+Stroke()}} while node position can be set using set\+Dag\+Node\+Pos(). ~\newline
 It\textquotesingle{}s possible to clone spline using clone().
