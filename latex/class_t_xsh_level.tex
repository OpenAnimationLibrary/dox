\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_level}{TXsh\+Level}} class is the base class of level managers in xsheet. 

Inherits {\bfseries{\mbox{\hyperlink{class_t_smart_object}{TSmart\+Object}}}} and {\bfseries{\mbox{\hyperlink{class_t_persist}{TPersist}}}}. ~\newline
 This is an abstract base class inherited by the concrete classes {\bfseries{\mbox{\hyperlink{class_t_xsh_child_level}{TXsh\+Child\+Level}}}}, {\bfseries{\mbox{\hyperlink{class_t_xsh_simple_level}{TXsh\+Simple\+Level}}}} and {\bfseries{\mbox{\hyperlink{class_t_xsh_zerary_fx_level}{TXsh\+Zerary\+Fx\+Level}}}}.

The class contains all features about a generic level and gives all methods to access to these information.

It\textquotesingle{}s possible ot know level name, \mbox{\hyperlink{class_t_xsh_level_a3a59fb1bd3261baee4e889a9edbf0ae8}{get\+Name()}}, or identify level types \mbox{\hyperlink{class_t_xsh_level_a1a147f1e970a62120c0bbaf0020a8932}{get\+Type()}}, defined in {\bfseries{\mbox{\hyperlink{}{txshleveltypes.\+h}}}}, and change it, \mbox{\hyperlink{class_t_xsh_level_ad2cb5812d9f36e9f5d570874dad8d3bd}{set\+Type()}}. The class provides function to know scene to which level belongs, \mbox{\hyperlink{class_t_xsh_level_a06176566e80858dbb206112db08a92b9}{get\+Scene()}}, and function to change scene \mbox{\hyperlink{class_t_xsh_level_a3c0052e5b6496d5bd9c697f678761e5c}{set\+Scene()}}.
