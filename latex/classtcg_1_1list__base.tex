\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class tcg\+::list\+\_\+base$<$ T $>$}
Base class for list-\/like containers based on {\ttfamily \mbox{\hyperlink{classtcg_1_1list}{tcg\+::list}}}\textquotesingle{}s indexed list implementation. 

This class implements a barebone container class which manages a collection of nodes stored in an iternal random access container.

Nodes management is limited to the {\itshape acquisition} and {\itshape release} of individual nodes (see the buy\+Node() and sell\+Node() functions).

Links between list nodes must be managed externally.

\begin{DoxySeeAlso}{See also}
Template class {\ttfamily \mbox{\hyperlink{classtcg_1_1list}{tcg\+::list}}} for an explanation of the indexed \mbox{\hyperlink{classtcg_1_1list}{list}} container concept. 
\end{DoxySeeAlso}
