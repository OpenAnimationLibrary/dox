\doxysubsection{Detailed Description}
The Mesh\+Texturizer-\/owned data about a texture. 

The purpose of \mbox{\hyperlink{class_mesh_texturizer}{Mesh\+Texturizer}} is that of storing in VRAM a set of textures cut up into tiles. This structure holds the data about the end-\/product of texture loading into a \mbox{\hyperlink{class_mesh_texturizer}{Mesh\+Texturizer}} instance.
