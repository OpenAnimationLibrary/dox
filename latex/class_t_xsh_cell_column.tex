\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_cell_column}{TXsh\+Cell\+Column}} class is the base class of column cell managers in xsheet.

Inherits {\bfseries{\mbox{\hyperlink{class_t_xsh_column}{TXsh\+Column}}}}. ~\newline
 This is an abstract base class inherited by the concrete classes {\bfseries{\mbox{\hyperlink{class_t_xsh_level_column}{TXsh\+Level\+Column}}}} and {\bfseries{\mbox{\hyperlink{class_t_xsh_zerary_fx_column}{TXsh\+Zerary\+Fx\+Column}}}}.

The class defines column by cells \mbox{\hyperlink{class_t_xsh_cell_column_ae349a5020445272a836bfa6c0368df3c}{get\+Cell\+Column()}}. \mbox{\hyperlink{class_t_xsh_cell_column}{TXsh\+Cell\+Column}} is an object composed of a {\bfseries{\mbox{\hyperlink{class_t_xsh_cell}{TXsh\+Cell}}}} vector and of an integer to memorize first not empty cell.

Class allows to manage cells in a column. It\textquotesingle{}s possible to know if cell is empty \mbox{\hyperlink{class_t_xsh_cell_column_abb7f749ea22c1114876f72e88e83a693}{is\+Cell\+Empty()}}, if column is empty \mbox{\hyperlink{class_t_xsh_cell_column_ae2c7bdf562fc2147a38d13b2b3ceb5cd}{is\+Empty()}}, to know column range \mbox{\hyperlink{class_t_xsh_cell_column_ab684089dcfd22454e7cbad249a04e29c}{get\+Range()}}, row count \mbox{\hyperlink{class_t_xsh_cell_column_a61c3721599568e5078baeaa2775d9016}{get\+Row\+Count()}}, last and first not empty frame \mbox{\hyperlink{class_t_xsh_cell_column_abb9061ec897fb594f9a8a580b6e36d9e}{get\+Max\+Frame()}}, \mbox{\hyperlink{class_t_xsh_cell_column_ad70a850bd9a11c11660a38d11559690f}{get\+First\+Row()}}. Specally class allows to change cells, \mbox{\hyperlink{class_t_xsh_cell_column_ac977346c15e229ba4ab069d6bb2d82f8}{get\+Cells()}}, \mbox{\hyperlink{class_t_xsh_cell_column_a91611c268308aa7d1dc8c4ebdcba43aa}{set\+Cells()}}, or individually cell, \mbox{\hyperlink{class_t_xsh_cell_column_aa747a68efe54d336d5d07dd8af451908}{get\+Cell()}}, \mbox{\hyperlink{class_t_xsh_cell_column_a7dc6cd2fa2a50088c4168a02dca582c1}{set\+Cell()}}; allows to insert \mbox{\hyperlink{class_t_xsh_cell_column_ac256660fc32543ab27d3e56149ca328d}{insert\+Empty\+Cells()}}, remove \mbox{\hyperlink{class_t_xsh_cell_column_a9dd9284327bd2bab258ce40f47a7b751}{remove\+Cells()}}, clear \mbox{\hyperlink{class_t_xsh_cell_column_a2e143e05d7c869fb49b3c7535984b2c2}{clear\+Cells()}} and move move\+Cells() cells in a column of xsheet.
