\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_frame_navigator}{Frame\+Navigator}} is a simple toolbar widget showing a numerical representation of an integer timeline. 

A \mbox{\hyperlink{class_frame_navigator}{Frame\+Navigator}} is a \mbox{\hyperlink{class_t_frame_handle}{TFrame\+Handle}} viewer composed of a line edit widget showing the current frame number, and a pair of directional arrow buttons that move to adjacent frames.

\begin{DoxyRemark}{Remarks}
The text field actually visualizes the internal integer {\ttfamily m\+\_\+frame} {\itshape +1} (e.\+g. when {\ttfamily m\+\_\+frame} is 0 then 1 is visualized). 
\end{DoxyRemark}
