\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_render_resource_manager}{TRender\+Resource\+Manager}} is a base class used to develop resource handlers for Toonz rendering contexts. {\itshape Resources} are hereby intended as data which possess special scope and duration with respect to one or more rendering processes. A most notable example of this is the internal Fxs cache, which is specialized to maintain a group of intermediate rendered tiles which may be shared in input by multiple fxs. ~\newline
 ~\newline
Creation of a resource manager requires the implementation of both the manager itself, and that of a \mbox{\hyperlink{class_t_render_resource_manager_generator}{TRender\+Resource\+Manager\+Generator}} class which is used internally to allocate manager instances at the most appropriate times. The manager\textquotesingle{}s generator needs to be instanced at {\bfseries{ file scope \texorpdfstring{$<$}{<}{\bfseries{}} \texorpdfstring{$>$}{>} to be correctly registered by Toonz. ~\newline
 ~\newline
Observe that the default scope of a resource manager is limited to a \mbox{\hyperlink{class_t_renderer}{TRenderer}} instance; specifically meaning that the swatch viewer, preview fx command, camstand preview and render/preview commands build different managers. }}
