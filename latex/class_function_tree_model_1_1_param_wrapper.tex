\doxysubsection{Detailed Description}
The common class representing a {\itshape parameter} in the model. 

\begin{DoxyRemark}{Remarks}
This class\textquotesingle{}s concept is different from that of a \mbox{\hyperlink{class_function_tree_model_1_1_channel}{Channel}}, as a {\itshape parameter} could be composed of {\itshape multiple channels}, e.\+g. like an animated {\ttfamily RGBA} color, which has 4 channels. 
\end{DoxyRemark}
