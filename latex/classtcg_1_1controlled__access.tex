\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Obj, typename Func = tcg\+::empty\+\_\+type$>$\newline
class tcg\+::controlled\+\_\+access$<$ Obj, Func $>$}
The \mbox{\hyperlink{classtcg_1_1controlled__access}{controlled\+\_\+access}} template class encapsulate an object that can be accessed {\bfseries{only}} after a functor has been invoked, enforcing access tracking at compile time. 

The accessor functor can be specified by explicit template argument or supplying it during access. In case the accessor type is specified, an instance of the functor will be stored together with the accessed variable.
