\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_t_thread_1_1_worker}{Worker}} is a specialized QThread that continuously polls \mbox{\hyperlink{class_t_thread_1_1_runnable}{Runnable}} tasks from a global execution queue to make them work.
