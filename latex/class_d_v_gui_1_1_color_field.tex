\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_color_field}{Color\+Field}} class is used to view an object to manage a color. 

\begin{DoxyVerb}         Inherits \b QWidget.

         The object is composed of a horizontal layout \b QHBoxLayout
\end{DoxyVerb}
 which contains a \mbox{\hyperlink{class_d_v_gui_1_1_style_sample}{Style\+Sample}}, and three or four \mbox{\hyperlink{class_d_v_gui_1_1_channel_field}{Channel\+Field}}, it depend if transparency is count, to manage color channel. You can pass to constructor current color value, \mbox{\hyperlink{class_d_v_gui_1_1_color_field_a6955ee5be0a395075590a99bc9262fe1}{get\+Color()}}, or set it calling set\+Color(). You can also pass to constructor a boolean to know if manage transparency channel or not, and an integer for \mbox{\hyperlink{class_d_v_gui_1_1_style_sample}{Style\+Sample}} size. \begin{DoxyVerb}         To know when color value change class provides a signal,
\end{DoxyVerb}
 color\+Changed(const TPixel32 \&); class emit signal when one \mbox{\hyperlink{class_d_v_gui_1_1_channel_field}{Channel\+Field}} value change. See SLOT\+: on\+Red\+Channel\+Changed(int value), on\+Green\+Channel\+Changed(int value), on\+Blue\+Channel\+Changed(int value) and on\+Alpha\+Channel\+Changed(int value) to know when signal is emitted.
