\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_palette_level}{TXsh\+Palette\+Level}} class represents a palette uncoupled from a level. 

Inherits {\bfseries{\mbox{\hyperlink{class_t_xsh_level}{TXsh\+Level}}}}. ~\newline
 A palette can be loaded in the xsheet and subsequently can be used as \char`\"{}external palette\char`\"{}. The number of frames of the palette-\/level depends on the palette animation.
