\doxysubsection{Detailed Description}
This file contains the implementation of a Mesh Deformer as specified in the Igarashi-\/\+Moscovich-\/\+Hughes paper \char`\"{}\+As-\/\+Rigid-\/\+As-\/\+Possible Shape Manipulation\char`\"{}, with some slight modifications. ~\newline
~\newline
Notably, the choice of handle points is not limited to mesh vertices, but free along the mesh surface. This can be achieved by constraining the classical least-\/squares minimization problems on a linear hyperplane, in the form\+:

$ \left{ \begin{array}{c} v^t H v + v^t f + c \rightarrow \mbox{min} \\ A v + d = 0 \end{array} \right. $ 

which can be solved using the Lagrange Multipliers theorem\+:

$ \Rightarrow \left{ \begin{array}{c} H v + f = A^t \lambda A v + d = 0 \end{array} \right. \Rightarrow \left( \begin{array}{c} H \\ A \end{array} \begin{array}{c} -A^t \\ 0 \end{array} \right) \left( \begin{array}{c} v \\ \lambda \end{array} \right) = \left( \begin{array}{c} -f \\ -d \end{array} \right) $
