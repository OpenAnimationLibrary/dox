\doxysubsection{Detailed Description}
Defines built-\/in Toonz zerary fxs. 

\begin{DoxyParagraph}{Toonz Internals -\/ Zerary Column Fxs}

\end{DoxyParagraph}
Toonz Zerary fxs are currently implemented with associated {\itshape  zerary column fxs \texorpdfstring{$<$}{<}\textbackslash{}I\texorpdfstring{$>$}{>} owning them. It is important to make these column fxs accessible from the actual zerary fx in order to allow complete traversability of an fxs tree. }
