\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_file_field}{File\+Field}} class provides an object to manage file browser. 

\begin{DoxyVerb}         Inherits \b QWidget.

         The object FileField is composed of two part, a field \b
\end{DoxyVerb}
 \mbox{\hyperlink{class_d_v_gui_1_1_line_edit}{Line\+Edit}} and a button {\bfseries{QPush\+Button}}. Click button to open a directory browser popup that is used to choose a directory. You can set in constructor default path. Maximum height object is fixed to {\bfseries{DVGui\+::\+Widget\+Height}}. By default dialog permit user to select only folder, but using \mbox{\hyperlink{class_d_v_gui_1_1_file_field_aa7a756fa242d0b6dbcf80994ee59c79e}{set\+File\+Mode()}} you can indicate what the user may select in the file dialog, folder or file; you can also set file type using \mbox{\hyperlink{class_d_v_gui_1_1_file_field_aad79aa2cfdb79a5e1a1753f3fefb7957}{set\+Filters()}}.
