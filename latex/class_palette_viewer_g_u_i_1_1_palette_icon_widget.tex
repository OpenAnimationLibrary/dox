\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_palette_viewer_g_u_i_1_1_palette_icon_widget}{Palette\+Icon\+Widget}} class provides a widget to show a palette icon. 

\begin{DoxyVerb} \brief    Special placeholder toolbar icon for \a starting a palette
\end{DoxyVerb}
 move through drag \& drop.

This widget is currently employed as a mere mouse event filter to propagate drag \& drop starts to a \mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}} ancestor in the widgets hierarchy.
