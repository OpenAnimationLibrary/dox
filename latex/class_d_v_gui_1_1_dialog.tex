\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_dialog}{Dialog}} class is the base class of dialog windows. 

Inherits {\bfseries{QDialog}}.

A dialog window is a top-\/level window, defined in {\bfseries{QDialog}}, this class provides a collection of functions to manage {\bfseries{QDialog}} widget visualization. \mbox{\hyperlink{class_d_v_gui_1_1_dialog}{Dialog}} window can be modal or modeless, from this it depends dialog visualization. You set your dialog to modal or modeless directly in constructor, with {\bfseries{bool}} has\+Button, in fact if the dialog has button then is modal too.

If dialog is {\bfseries{modal}} it\textquotesingle{}s composed of two part, two {\bfseries{QFrame}}; the first, {\bfseries{main}} {\bfseries{part}} at top of dialog, contains a vertical box layout {\bfseries{QVBox\+Layout}}, the second, {\bfseries{button}} {\bfseries{part}} at bottom, contains an horizontal box layout {\bfseries{QHBox\+Layout}}. Else if dialog is modeless it is composed only of first part, to be more precise, it is composed of a {\bfseries{QFrame}} containing a vertical box layout {\bfseries{QVBox\+Layout}}. \begin{DoxyVerb}            The first part of dialog can be implemented using different
\end{DoxyVerb}
 objects. You can manage directly vertical box layout, insert widget \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ada55b78010dbe97d4f2692cc6f2e933f}{add\+Widget()}}, or layout \mbox{\hyperlink{class_d_v_gui_1_1_dialog_adecc035c55394875b6354480911feab2}{add\+Layout()}}, set border and spacing, using \mbox{\hyperlink{class_d_v_gui_1_1_dialog_a7aa8ccf399f483fe1f36c30a1318987a}{set\+Top\+Margin()}} and \mbox{\hyperlink{class_d_v_gui_1_1_dialog_af1334e3123db7e7c9a9b90ee7c73e226}{set\+Top\+Spacing()}}, or insert a spece, \mbox{\hyperlink{class_d_v_gui_1_1_dialog_af739c2e47fec5f24842391da54fe7917}{add\+Spacing()}}. ~\newline
 If you need an horizontal layout you can use an implemented horizontal box layout; the function \mbox{\hyperlink{class_d_v_gui_1_1_dialog_a21628a44d453352094b61b62b4407c6b}{begin\+HLayout()}} initialize the horizontal box layout, a collection of function permit to insert object in this layout; at the end you must recall \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ab88d88334402845e544da6ca1f207c79}{end\+VLayout()}}, this add horizontal layout to first part of dialog. This struct permit you to align the object you insert. ~\newline
 If you need vertical layout you can use an implemented vertical box layout composed of two column, the function \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ae7de609b3b23233656bc68852a2d10cb}{begin\+VLayout()}} initialize the vertical box layout, a collection of function permit to insert object in this layout. All this functions insert a pair of object in two vertical layout and permit to align the objects, all pairs you insert are tabulated. At the end you must recall \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ab88d88334402845e544da6ca1f207c79}{end\+VLayout()}}, this add vertical layout to first part of dialog.

The second part of dialog may be implemented if dialog is modal. It\textquotesingle{}s possible insert one, two or three widget in its horizontal box layout using \mbox{\hyperlink{class_d_v_gui_1_1_dialog_aff324b24a1ad0c06fdc178024c1f0cf0}{add\+Button\+Bar\+Widget()}}, or clear all widget using \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ac1877362599959156653d817b7cb7288}{clear\+Button\+Bar()}}. You can also set margin and spacing with \mbox{\hyperlink{class_d_v_gui_1_1_dialog_a5ea3d0ab4b8209de419f5463d7145271}{set\+Button\+Bar\+Margin()}} and \mbox{\hyperlink{class_d_v_gui_1_1_dialog_ad9b1d78f491c3f92c0d0503e04a8917a}{set\+Button\+Bar\+Spacing()}} functions.
