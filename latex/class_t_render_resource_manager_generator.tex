\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_render_resource_manager_generator}{TRender\+Resource\+Manager\+Generator}} class is used to construct resource managers that will be used by TRenderers. Please refer to the \mbox{\hyperlink{class_t_render_resource_manager}{TRender\+Resource\+Manager}} documentation for the details.

\begin{DoxyNote}{Note}
Resource managers may be constructed with a {\itshape  render instance \texorpdfstring{$<$}{<}\textbackslash{}i\texorpdfstring{$>$}{>} scope rather than the default render context (ie \mbox{\hyperlink{class_t_renderer}{TRenderer}}) scope. This can be achieved by passing {\ttfamily true} at the generator\textquotesingle{}s constructor, proving especially useful considering that the same \mbox{\hyperlink{class_t_renderer}{TRenderer}} may virtually start multiple render instances at the same time. }
\end{DoxyNote}
