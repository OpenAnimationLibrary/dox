\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_channel_field}{Channel\+Field}} class is used to view an object to manage a color value, red, green, blue or transparency channel. 

\begin{DoxyVerb}         Inherits \b QWidget.

         The object is composed of grid layout \b QGridLayout which
\end{DoxyVerb}
 contains a label in first row, first column, to identify channel, a text field {\bfseries{\mbox{\hyperlink{class_d_v_gui_1_1_int_line_edit}{Int\+Line\+Edit}}}} in first row, second column, and a slider in second row, second column. Texf field and slider are connected, so if you change one the other automatically change. You can set current value \mbox{\hyperlink{class_d_v_gui_1_1_channel_field_ae5755f1cebca40ceb487e0e0bff312bc}{get\+Channel()}}, using \mbox{\hyperlink{class_d_v_gui_1_1_channel_field_ad553ee5e2c0fe57d1bd1e8626648aeff}{set\+Channel()}}. This two object is used to manage channel value, them range is fixed to \mbox{[}0,255\mbox{]}. This object size is fixed, \mbox{[}50, 2$\ast$\+DVGui\+::\+Widget\+Height\mbox{]}.

To know when channel parameter value change class provides a signal, value\+Changed(int value); class emit signal when slider value change or when text field is editing, see SLOT\+: \mbox{\hyperlink{class_d_v_gui_1_1_channel_field_a36098f7136611a94c5cc487e71b95e1b}{on\+Slider\+Changed(int value)}} and \mbox{\hyperlink{class_d_v_gui_1_1_channel_field_a3ccf5ccfc4c97c2f661e6023f20ba870}{on\+Edit\+Changed(const \+QString \&str)}} to know when signal is emitted.
