\doxysubsection{Detailed Description}
This file contains an implementation of a greyscale (ie per-\/channel) erode/dilate morphological operator, following the van Herk/\+Gil-\/\+Werman O(row$\ast$cols) algorithm.

An extension with circular structuring element is attempted -\/ unfortunately I could not retrieve a copy of Miyataka\textquotesingle{}s paper about that, which seemingly claimed O(rows $\ast$ cols) too. The implemented algorithm is a sub-\/optimal O(rows$\ast$cols$\ast$radius).
