\subsubsection*{template$<$typename Ran\+It$>$\newline
class tcg\+::polyline\+\_\+ops\+::\+Standard\+Deviation\+Evaluator$<$ Ran\+It $>$}
Calculates the (weighted) standard deviation of a polyline\textquotesingle{}s sub-\/paths with respect to the segment connecting the endpoints.

\begin{DoxyWarning}{Warning}
For efficiency reasons, the returned value is the actual standard deviation, times the endpoints-\/segment length. 
\end{DoxyWarning}
