\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_res_list_manager}{Res\+List\+Manager}} is a {\itshape singleton} class which manages presets of camera resolutions.

This class allows a lightweight management of the resolution presets used in camera settings. Specific tasks include resolution fetching and the add/remove functions. Adding or removing presets causes the list\+Changed() signal to be emitted.

\begin{DoxyNote}{Note}
There actually are {\itshape two} distinct instances of this class, one managing standard cameras and the other for cleanup cameras. Presets are stored in two distinct files as well. 
\end{DoxyNote}
