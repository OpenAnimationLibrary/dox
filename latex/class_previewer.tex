\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_previewer}{Previewer}} class is a singleton which deals with Preview renders of isolated frames. The main purpose of this class, as of Toonz 6.\+x, is that of managing preview requests coming from Toonz\textquotesingle{}s Scene Viewers; however, it is designed to support requests from a generic {\bfseries{\mbox{\hyperlink{class_previewer_1_1_listener}{Previewer\+::\+Listener}}}} class. The \mbox{\hyperlink{class_previewer}{Previewer}} is a singleton, meaning that it centralizes requests from many possible listeners in a single collection of previewed frames. ~\newline
 ~\newline
The most important method of the class is {\bfseries{\mbox{\hyperlink{class_previewer_afb0f0a5c7ca1fb9366996407ea58d4cc}{get\+Raster()}}}}, which attempts retrieval of a certain frame and, if the frame was not calculated yet, makes the \mbox{\hyperlink{class_previewer}{Previewer}} build it. ~\newline
The \mbox{\hyperlink{class_previewer_a4f2238933e3e040b9c568525d487dbde}{update()}} slot is provided to refresh the cached information about the entire collection of currently stored frames. Updated frames are recalculated only upon a new \mbox{\hyperlink{class_previewer_afb0f0a5c7ca1fb9366996407ea58d4cc}{get\+Raster()}}, in case the previously stored information no more match the current frame description. ~\newline
The clear() methods make the \mbox{\hyperlink{class_previewer}{Previewer}} erase all stored information about one or all frames, so that the following \mbox{\hyperlink{class_previewer_afb0f0a5c7ca1fb9366996407ea58d4cc}{get\+Raster()}} will forcibly recalculate the requested frame.

The suspend\+Rendering method allows suspension of the previewer\textquotesingle{}s rendering activity for safety purposes, typically related to the fact that no rendering process should actually be performed as the underlying scene is about to change or being destroyed. Upon suspension, further rendering requests are silently rejected -\/ and currently active ones are canceled and waited until they are no longer active. NOTE\+: This method is currently static declared, since the \mbox{\hyperlink{class_previewer}{Previewer}} must be be instanced only after a consistent scene has been initialized. This method is allowed to bypass such limitation.
