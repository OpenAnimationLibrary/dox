\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_t_dock_widget}{TDock\+Widget}} class is a convenience QDock\+Widget-\/like interface to \mbox{\hyperlink{class_dock_widget}{Dock\+Widget}} class. It provides an internal base layout including a titlebar and a content widget.
