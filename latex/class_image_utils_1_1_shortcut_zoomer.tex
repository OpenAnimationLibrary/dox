\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_image_utils_1_1_shortcut_zoomer}{Shortcut\+Zoomer}} abstract base class is used by viewer widget to access shortcut-\/related commands. 

This class is a wrapper for shortcuts established by the \mbox{\hyperlink{class_command_manager}{Command\+Manager}} interface.

Subclass it defining the required view commands, then implement a {\ttfamily key\+Press\+Event()} event handler in the viewer widget you want\+:
