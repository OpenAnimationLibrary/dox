\doxysubsection{Detailed Description}
This is the base class for the different tags that form the PLI format. For an explanation of each one of them, refer to the documentation The different tags can be extracted fronm the class \mbox{\hyperlink{class_parsed_pli}{Parsed\+Pli}}, using methods get\+First\+Tag and get\+Next\+Tag. The tag type is stored in m\+\_\+type member.
