\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_output_properties}{TOutput\+Properties}} class provides a container for output properties and gives all methods to access to these information.

The class contains all features necessaries to compute output and provides a collection of functions that return the various feature, and enable manipulation of these.

A path \mbox{\hyperlink{class_t_output_properties_ab082a7e84177a558484d648215e4b4f7}{get\+Path()}} is used to manipulate output name and directory, path can be changed using the \mbox{\hyperlink{class_t_output_properties_abd2cedbcd56198c5d6e9e9ffcd51767e}{set\+Path()}}.

The \mbox{\hyperlink{class_t_output_properties_aae127cbd8b19eaed0b0fbaf589349176}{get\+Range()}} function tells whether is set a range in output, if it\textquotesingle{}s so provides first frame, last frame, and step; this element can be changed using \mbox{\hyperlink{class_t_output_properties_a3673546179602582d1375ba6961ceea6}{set\+Range()}}.

The \mbox{\hyperlink{class_t_output_properties_a0d9662971bc33097aa1b55111965b967}{get\+Frame\+Rate()}} function return output frame rate, it can be set by \mbox{\hyperlink{class_t_output_properties_ad88500532eb3bd83be26ae6ed256adcd}{set\+Frame\+Rate()}}.

The \mbox{\hyperlink{class_t_output_properties_a8f0571bf6bba6dcabe2f3891936b2059}{get\+Which\+Levels()}} function return a value is used to characterize which levels are in output\+:

\begin{DoxyItemize}
\item If value is TOutput\+Properties\+::\+All\+Levels all levels are computed in output. \item If value is TOutput\+Properties\+::\+Selected\+Only only selectecd levels are computed in output. \item If value is TOutput\+Properties\+::\+Animated\+Only only animated levels are computed in output.\end{DoxyItemize}
It can be set by \mbox{\hyperlink{class_t_output_properties_af44dcec523fb8f5dc2ad56b3e74860a2}{set\+Which\+Levels()}}.

The get\+File\+Format\+Properties(string ext) function return file of format {\bfseries{ext}} properties {\bfseries{\mbox{\hyperlink{class_t_property_group}{TProperty\+Group}}}}. It\textquotesingle{}s possibile set different format and have appropriate properties.

The \mbox{\hyperlink{class_t_output_properties_afa7c771452b57b50047aa4afc6032cad}{get\+Render\+Settings()}} function return render settings {\bfseries{\mbox{\hyperlink{class_t_render_settings}{TRender\+Settings}}}}, it can be change using \mbox{\hyperlink{class_t_output_properties_a33595887e47b7bb0c988420636763731}{set\+Render\+Settings()}}.
