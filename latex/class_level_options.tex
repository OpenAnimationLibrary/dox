\doxysubsection{Detailed Description}
User-\/editable level properties. 

Historically, \mbox{\hyperlink{class_level_properties}{Level\+Properties}} stores properties both accessible to Toonz end-\/users, and internal to the Toonz application.

The user-\/accessible options have been moved to this nested class to clearly separate them from internal properties of a level.
