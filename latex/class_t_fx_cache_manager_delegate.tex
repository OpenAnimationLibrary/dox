\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_t_fx_cache_manager_delegate}{TFx\+Cache\+Manager\+Delegate}} is the base class to implement cache managers used in Toonz render processes.

A \mbox{\hyperlink{class_t_fx_cache_manager_delegate}{TFx\+Cache\+Manager\+Delegate}} instance automatically receives access notifications through the \mbox{\hyperlink{class_t_fx_cache_manager}{TFx\+Cache\+Manager}} associated with a render process. These may either concern a request of a certain cache resource, or the storage proposal of a calculated node result.

\begin{DoxyWarning}{Warning}
Delegates are expected to be dependent on \mbox{\hyperlink{class_t_fx_cache_manager}{TFx\+Cache\+Manager}} -\/ use the MANAGER\+\_\+\+FILESCOPE\+\_\+\+DECLARATION\+\_\+\+DEP macro to achieve that.
\end{DoxyWarning}
