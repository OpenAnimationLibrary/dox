\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_studio_palette_viewer}{Studio\+Palette\+Viewer}} class provides an object to view and manage studio palettes. 

The \mbox{\hyperlink{class_studio_palette_viewer}{Studio\+Palette\+Viewer}} class provides an object to view and manage studio palettes.

Inherits {\bfseries{QFrame}}. This object is composed of a splitter {\bfseries{QSplitter}} that contain a vertical layout and a {\bfseries{\mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}}}}. Vertical layout contain a {\bfseries{\mbox{\hyperlink{class_studio_palette_tree_viewer}{Studio\+Palette\+Tree\+Viewer}}}} and a toolbar, this object allows to manage the palettes in studio palette folders. {\bfseries{\mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}}}} is set to fixed view type\+: {\bfseries{\mbox{\hyperlink{namespace_palette_viewer_g_u_i_a8198a2523c81b8ac94625e531aa0a789a5f5db8a4e29bfbde9427776e7268888d}{Palette\+Viewer\+GUI\+::\+STUDIO\+\_\+\+PALETTE}}}} allows to show and modify current studio palette selected in tree.

Inherits {\bfseries{QFrame}}. This object is composed of a splitter {\bfseries{QSplitter}} that contain a vertical layout and a {\bfseries{\mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}}}}. Vertical layout contain a {\bfseries{\mbox{\hyperlink{class_studio_palette_tree_viewer}{Studio\+Palette\+Tree\+Viewer}}}} and a toolbar, this object allows to manage the palettes in studio palette folders. {\bfseries{\mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}}}} is set to fixed view type\+: {\bfseries{\mbox{\hyperlink{namespace_palette_viewer_g_u_i_a8198a2523c81b8ac94625e531aa0a789a5f5db8a4e29bfbde9427776e7268888d}{Palette\+Viewer\+GUI\+::\+STUDIO\+\_\+\+PALETTE}}}} allows to show and modify current studio palette selected in tree.
