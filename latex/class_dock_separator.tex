\doxysubsection{Detailed Description}
Separators are interposition widgets among docked Dock\+Widgets of a \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}}.

A \mbox{\hyperlink{class_dock_separator}{Dock\+Separator}} has the role of separating sister Regions; it is always owned by a parent \mbox{\hyperlink{class_region}{Region}} and inherits its subdivision (here separation) direction. It also provides basical interaction with the user, allowing itself to be shifted along separation direction until geometric constraints are met. \mbox{\hyperlink{class_dock_separator}{Dock\+Separator}} class can be inherited to build custom separators -\/ in that case, a heir of \mbox{\hyperlink{class_dock_deco_allocator}{Dock\+Deco\+Allocator}} class allocating the new separator class must be assigned to the \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}}. It is also possible to specify the Separators\textquotesingle{} thickness used in a \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}} through the QLayout\+::set\+Spacing(int) method.

{\bfseries{NOTE\+:}} Observe that Separators\textquotesingle{} geometry is owned by the \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}} to which it belongs; it is discouraged (but not forbidden) to manually modify it. You may, for example, modify the geometry of a \mbox{\hyperlink{class_dock_separator}{Dock\+Separator}} when dragging a dock widget over it. In any case, the layout will automatically regenerate Separators\textquotesingle{} geometry at each update of the layout.
