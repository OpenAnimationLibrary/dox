\doxysubsection{Detailed Description}
This file contains the barebone of an elementary, classical implementation of the Observer pattern requiring derivation. 

This is intended as a convenient and lightweight predefined implementation when hardwiring the Observer pattern in a known class using derivation.

Observe that this implementation {\itshape does not deal with the actual notifications forwarding} -\/ it just manages the set of observers/notifiers in a notifier/observer instance.

No multithreading is taken into consideration.
