\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_affine_fx}{Affine\+Fx}} is a specialization of \mbox{\hyperlink{class_t_geometry_fx}{TGeometry\+Fx}} which implements animated or stage-\/controlled affines

This specific implementation of \mbox{\hyperlink{class_t_geometry_fx}{TGeometry\+Fx}} is needed to deal with those affines which are best {\bfseries{not}} resolved during the rendering-\/tree expansion procedure.
