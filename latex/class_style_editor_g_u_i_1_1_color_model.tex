\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_style_editor_g_u_i_1_1_color_model}{Color\+Model}} provides an object to manage color change and its transformation from rgb value to hsv value and vice versa. 

This object change color using its rgb channel or its hsv channel; if you change a color channel class assure you that other channel not change.
