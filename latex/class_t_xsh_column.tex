\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_column}{TXsh\+Column}} class is the base class of column managers in xsheet. 

Inherits {\bfseries{\mbox{\hyperlink{class_t_column_header}{TColumn\+Header}}}} and {\bfseries{\mbox{\hyperlink{class_t_persist}{TPersist}}}}. ~\newline
 This is an abstract base class inherited by the concrete classes {\bfseries{\mbox{\hyperlink{class_t_xsh_cell_column}{TXsh\+Cell\+Column}}}}, {\bfseries{\mbox{\hyperlink{class_t_xsh_sound_column}{TXsh\+Sound\+Column}}}} and \mbox{\hyperlink{class_t_xsh_palette_column}{TXsh\+Palette\+Column}}.

The class contains all features about a generic xsheet column and gives all methods to access to these information.

The \mbox{\hyperlink{class_t_xsh_column_aee2af1e0385299fbf26869f2f633a1d7}{create\+Empty()}} static method creates an new empty column.

The \mbox{\hyperlink{class_t_xsh_column_a2d812ed804dba4d8d91482a5f35e2601}{get\+Xsheet()}} function return xsheet to which column belongs, xsheet can be changed using the \mbox{\hyperlink{class_t_xsh_column_ad4badbad6279671e8a42037873647de9}{set\+Xsheet()}}.

The \mbox{\hyperlink{class_t_xsh_column_a50b10129143f025fd95d4a424cd288ac}{get\+Icon()}} function return column icon.

\mbox{\hyperlink{class_t_xsh_column}{TXsh\+Column}} allows to manage generic column status \mbox{\hyperlink{class_t_xsh_column_a93756d61d42ea3125e417172cc0481a1}{get\+Status\+Word()}} and set\+Status\+Word. Particularly it\textquotesingle{}s possible to know if column is a mask \mbox{\hyperlink{class_t_xsh_column_a84fb2d5883fcc82a5f3aab2f8bb8b495}{is\+Mask()}}, is lock \mbox{\hyperlink{class_t_xsh_column_a04953896aae8860ace46010956a28db9}{is\+Locked()}}, or if it\textquotesingle{}s view is in modality preview or camera stand, \mbox{\hyperlink{class_t_xsh_column_ae8f2989de835d8ed4ca4c246f70127f1}{is\+Camstand\+Visible()}} and \mbox{\hyperlink{class_t_xsh_column_aba9255d82fc5bbc87f126060d31d0da2}{is\+Preview\+Visible()}}. It\textquotesingle{}s besides possible ot change this settings using set\+Mask(), \mbox{\hyperlink{class_t_xsh_column_aa4fdf492c76dd277452ab9ec23cea6dc}{lock()}}, \mbox{\hyperlink{class_t_xsh_column_ad43e30c44ccfe83b71b6859d85af41e9}{set\+Camstand\+Visible()}} and \mbox{\hyperlink{class_t_xsh_column_a8d475967039cbeb4604ea438038c81e2}{set\+Preview\+Visible()}}.
