\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_int_field}{Int\+Field}} class provides to view an object to manage an integer parameter. 

Inherits {\bfseries{QWidget}}.

The class is composed of an horizontal layout which contains two object, \begin{DoxyItemize}
\item a vertical layout with a text field {\bfseries{\mbox{\hyperlink{class_d_v_gui_1_1_int_line_edit}{Int\+Line\+Edit}}}} and a roller \item an horizontal slider QSlider. This objects are connected, so if you change one the other automatically change. You can set current value \mbox{\hyperlink{class_d_v_gui_1_1_int_field_a5657b3a478f9e4993c7d48a5338dbb75}{get\+Value()}}, minimum and max value using \mbox{\hyperlink{class_d_v_gui_1_1_int_field_aed421e2c52946f2c7643534b4f6f13f7}{set\+Value()}}, \mbox{\hyperlink{class_d_v_gui_1_1_int_field_a055d473a8dde4481be33a83c9fa04c96}{set\+Range()}} or \mbox{\hyperlink{class_d_v_gui_1_1_int_field_a4739e5468df7fb725423b99ff66950b4}{set\+Values()}}; by default is value = 0, minimum value = 0 and maximum value = 100. ~\newline
 Maximum height object is fixed to {\bfseries{DVGui\+::\+Widget\+Height}}, width depend on parent width.\end{DoxyItemize}
To know when integer parameter value change class provides a signal, \mbox{\hyperlink{class_d_v_gui_1_1_int_field_a8bf65107aaa469d649e8b51297a5f838}{value\+Changed()}}; class emit signal when slider value change or when editing text field is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT\+: \mbox{\hyperlink{class_d_v_gui_1_1_int_field_ab9e93bb4afd724775e330fc4eb40acaf}{on\+Slider\+Changed(int value)}}, \mbox{\hyperlink{class_d_v_gui_1_1_int_field_ae612e83482e631a1eb4c7a41005941a7}{on\+Editing\+Finished()}}.
