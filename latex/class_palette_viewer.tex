\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_palette_viewer}{Palette\+Viewer}} class provides an object to view and manage palette view. 

Inherits {\bfseries{QWidget}}.

This object allows to show and manage palette; it\textquotesingle{}s possible distinguish three view type, class can show\+: current level palette, current studio palette or current cleanup palette, using set\+Palette() to set palette viewer. The object is composed of a vertical layout with three widget\+: \begin{DoxyItemize}
\item a tab bar {\bfseries{Palette\+Viewer\+GUI\+::\+Tab\+Bar}} in which are displayed palette page {\bfseries{\mbox{\hyperlink{class_palette_viewer_af5729b607b5acde1440650eaf124db53}{create\+Tab\+Bar()}}}}; \item a central frame {\bfseries{\mbox{\hyperlink{class_palette_viewer_g_u_i_1_1_page_viewer}{Palette\+Viewer\+GUI\+::\+Page\+Viewer}}}} in which are displayed all style of current page in palette \item a button bar, {\bfseries{\mbox{\hyperlink{class_palette_viewer_a2cd89a1c0a669f8efdf741cb7459357c}{create\+Tool\+Bar()}}}}, {\bfseries{\mbox{\hyperlink{class_palette_viewer_a4595848afbdf3dbfb42b559ca7a81121}{create\+Palette\+Tool\+Bar()}}}} and {\bfseries{\mbox{\hyperlink{class_palette_viewer_ab75f6d31f7b756936ce54f0b6abb73a9}{create\+Save\+Palette\+Tool\+Bar()}}}};\end{DoxyItemize}
A collection of method allows you to manage this object and its interaction with palette.
