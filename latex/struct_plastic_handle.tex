\doxysubsection{Detailed Description}
The \mbox{\hyperlink{struct_plastic_handle}{Plastic\+Handle}} class models the geometrical information of the neighbourhood of a point that can be deformed with a \mbox{\hyperlink{class_plastic_deformer}{Plastic\+Deformer}} instance.

The \mbox{\hyperlink{class_plastic_deformer}{Plastic\+Deformer}} class allows users to deform interactively a mesh object. A Plastic Handle models a point of the mesh that can be {\itshape manually} displaced by the user to deform the mesh object. ~\newline
~\newline
 It contains the initial position of the point to be displaced; eventually, further implementation may include a \textquotesingle{}{\itshape direction\texorpdfstring{$<$}{<}\textbackslash{}I\texorpdfstring{$>$}{>}\textquotesingle{} that can be rotated by the user to force the neighborhood into a specific rotational component. }
