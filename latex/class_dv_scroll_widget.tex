\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_dv_scroll_widget}{Dv\+Scroll\+Widget}} class is a QScroll\+Area-\/like container widget for monodirectional scrolling. 

Unlike a QScroll\+Area, scrolling is enforced using directional buttons at the side of the content whose activation performs animated scrolling by half a widget in the represented direction. Alternatively, free-\/scroll is available by default where mouse events are un-\/accepted by the content\textquotesingle{}s child widgets.

The push buttons at the side of the content appear only when there is available content space in that direction. They are {\itshape empty} by default, but can be filled in or styled with a style sheet. In the latter case, they are given an object name dependent on the scroll orientation -\/ in the form {\ttfamily Scroll\texorpdfstring{$<$}{<}Left/\+Right/\+Up/\+Down\texorpdfstring{$>$}{>}Button}.

\begin{DoxyNote}{Note}
The default size QSize\+Policy for \mbox{\hyperlink{class_dv_scroll_widget}{Dv\+Scroll\+Widget}} is {\ttfamily Preferred} in the orientation direction, and {\ttfamily Fixed} in the other.
\end{DoxyNote}
