\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_cell}{TXsh\+Cell}} class provides a cell of xsheet, this structure contains a level frame.

\mbox{\hyperlink{class_t_xsh_cell}{TXsh\+Cell}} is composed by a pointer to level {\bfseries{TXsh\+LevelP}} and a {\bfseries{\mbox{\hyperlink{class_t_frame_id}{TFrame\+Id}}}} that identifies right frame in level.

The class provides functions that return the cell element. The \mbox{\hyperlink{class_t_xsh_cell_a20b623d308bc09b820ec6b116516f564}{get\+Image()}} function return image associated to cell. The \mbox{\hyperlink{class_t_xsh_cell_a8898cc47eba24e59477bfba8e2511f1d}{get\+Simple\+Level()}} function return simple level {\bfseries{\mbox{\hyperlink{class_t_xsh_simple_level}{TXsh\+Simple\+Level}}}} to which cell belongs. The \mbox{\hyperlink{class_t_xsh_cell_a5a1965b96c2c698d6f0fb42a80c03406}{get\+Frame\+Id()}} function return {\bfseries{\mbox{\hyperlink{class_t_frame_id}{TFrame\+Id}}}} frame identification of cell. It\textquotesingle{}s possible to know if cell is empty through \mbox{\hyperlink{class_t_xsh_cell_a5d20229e7bcd19957b123959e58a833e}{is\+Empty()}}.
