\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_dummy_layout}{Dummy\+Layout}} class implements a layout whose items\textquotesingle{} geometry is not enforced at all, ignoring parent size bounds or geometry. 

A \mbox{\hyperlink{class_dummy_layout}{Dummy\+Layout}} serves as a simple, inert container of layout items. It does not enforce geometry to items added to it, allowing objects using this layout to appear as a \textquotesingle{}viewport\textquotesingle{} for free-\/geometry items.

\begin{DoxyNote}{Note}
As this layout does not effectively manage its objects\textquotesingle{} geometry, these must be managed outside. 
\end{DoxyNote}
