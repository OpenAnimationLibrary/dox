\doxysubsection{Detailed Description}
This is a private class used to convey the \mbox{\hyperlink{class_t_renderer_a478735dc2935569993603911e116e0f5}{TRenderer\+::start\+Rendering}} methods into Qt queued slots. This is necessary since these methods implicitly perform event processing, which could cause trouble in case they are invoked from events which must respect a strict ordering. ~\newline
 ~\newline
For example, suppose that a render must be invoked upon a mouse\+Press\+Event, and that such event must have been completely processed before the correspondent mouse\+Release\+Event is invoked -\/ calling the start\+Rendering method {\itshape directly} by the mouse\+Press\+Event may cause the mouse\+Release\+Event to be processed before the former\textquotesingle{}s end.
