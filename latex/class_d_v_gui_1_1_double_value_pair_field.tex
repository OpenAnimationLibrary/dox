\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field}{Double\+Value\+Pair\+Field}} class provides to view an object to manage a pair of double parameter. 

Inherits {\bfseries{QWidget}}.

The object is composed of an horizontal layout {\bfseries{QHBox\+Layout}} which contains two pair \mbox{[}label, text field\mbox{]} and a slider with two grab, one for each double value to manage. \mbox{[}label, text field\mbox{]} are a {\bfseries{QLabel}} and a {\bfseries{\mbox{\hyperlink{class_d_v_gui_1_1_double_line_edit}{Double\+Line\+Edit}}}}. Objects are inserted in the following order\+: a label and respective text field, the slider and the other pair \mbox{[}lebel,text field\mbox{]}. Object size width is not fixed, while height is equal to DVGui\+::\+Widget\+Height, 20; labels width depend from its text length, text fields has fixed size, while slider width change in according with widget size.

Objects contained in this widget are connected, so if you change one value the other automatically change, if it is necessary. You can set current value, \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_a133bf611c574ae12e1c588ced715c840}{get\+Values()}}, minimum and max value, \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_a8c03d2bce62a61deb9f8be4b82aaf206}{get\+Range()}}, using \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_adb89b5fa4f166bb8dad6f60416913cc4}{set\+Values()}}, \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_ac2d51974644cab91269a4986344b90b3}{set\+Range()}}. \begin{DoxyVerb}         To know when one of two double parameter value change class
\end{DoxyVerb}
 provides a signal, \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_a43a6a7b8c1db85d0c3b24594ef0b33e1}{values\+Changed()}}; class emit signal when a grab slider position change or when editing text field, left or right is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT\+: \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_ad5d535a3ee95e7c91d47ed1558120e7c}{on\+Left\+Editing\+Finished()}} and \mbox{\hyperlink{class_d_v_gui_1_1_double_value_pair_field_a957598bd7a4b9b8c0825230cb82cb52e}{on\+Right\+Editing\+Finished()}}.
