\doxysubsection{Detailed Description}
Dockable widgets are widget containers handled by \mbox{\hyperlink{class_dock_layout}{Dock\+Layout}} class. 

Dock\+Layouts accept only dock widgets of this class; so if you want to place a given widget under this kind of layout, a \mbox{\hyperlink{class_dock_widget}{Dock\+Widget}} shell for it must be allocated first. The following class implements base dock widgets, with native floating decorations and no docked title bar. It is encouraged to reimplement all the essential functions for custom aspect and behaviour.

\mbox{\hyperlink{class_dock_layout}{Dock\+Layout}} and \mbox{\hyperlink{class_dock_placeholder}{Dock\+Placeholder}} classes.
