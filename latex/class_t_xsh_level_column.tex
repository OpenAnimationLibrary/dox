\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_level_column}{TXsh\+Level\+Column}} class provides a column of levels in xsheet and allows its management.

Inherits {\bfseries{\mbox{\hyperlink{class_t_xsh_cell_column}{TXsh\+Cell\+Column}}}}. ~\newline
 The class defines column of levels \mbox{\hyperlink{class_t_xsh_level_column_a8d428cf082e9db15e8d73afa857beb79}{get\+Level\+Column()}}, more than {\bfseries{\mbox{\hyperlink{class_t_xsh_cell_column}{TXsh\+Cell\+Column}}}} has a pointer to {\bfseries{\mbox{\hyperlink{class_t_level_column_fx}{TLevel\+Column\+Fx}}}} \mbox{\hyperlink{class_t_xsh_level_column_a975990a16e1c3dc5cf4348908cac259b}{get\+Level\+Column\+Fx()}} and a {\bfseries{string}} to identify icon. The string is an icon identification helpful to level icon management, \mbox{\hyperlink{class_t_xsh_column_a50b10129143f025fd95d4a424cd288ac}{get\+Icon()}} update\+Icon().
