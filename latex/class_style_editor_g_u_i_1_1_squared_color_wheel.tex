\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_style_editor_g_u_i_1_1_squared_color_wheel}{Squared\+Color\+Wheel}} is a squared color to change color. 

\begin{DoxyVerb}        Inherits \b QWidget.

        This object show a square faded from one color channel to
\end{DoxyVerb}
 another color channel, the two channel represent x and y axis of square. It\textquotesingle{}s possible to choose viewed shade using {\bfseries{\mbox{\hyperlink{class_style_editor_g_u_i_1_1_squared_color_wheel_a27a141bc6ceecccf7b17251825c28196}{set\+Channel()}}}}. Click in square change current \mbox{\hyperlink{class_style_editor_g_u_i_1_1_squared_color_wheel}{Squared\+Color\+Wheel}}.
