\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_double_value_field}{Double\+Value\+Field}} class provides to view an object to manage a generic double parameter, with two cipher after decimal point. 

\begin{DoxyVerb}            Inherits \b QWidget.

            The class is composed of an horizontal layout which contains two
\end{DoxyVerb}
 object, \begin{DoxyItemize}
\item a vertical layout with a text field {\bfseries{Primary\+Double\+Line\+Edit}} and a roller \item an horizontal slider QSlider. This objects are connected, so if you change one the other automatically change. It\textquotesingle{}s possible to set current value get\+Value(), minimum and max value, get\+Range(), using set\+Value(), set\+Range() or \mbox{\hyperlink{class_d_v_gui_1_1_double_value_field_a76080d3059f984a91b993398468e09d7}{set\+Values()}}; by default value = 0, minimum value = -\/100.\+0 and maximum value = 100.\+0. ~\newline
 Maximum height object is fixed to {\bfseries{DVGui\+::\+Widget\+Height}}, while width depend on parent width.\end{DoxyItemize}
To know when double parameter value change class provides a signal, \mbox{\hyperlink{class_d_v_gui_1_1_double_value_field_a1bfae9d848d4f1a1560d850d42e2eb40}{value\+Changed()}}; class emit signal when slider value change or when editing text field is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT\+: \mbox{\hyperlink{class_d_v_gui_1_1_double_value_field_a3885689e08ced081cc592376803dc994}{on\+Slider\+Changed(int value)}}, on\+Editing\+Finished().
