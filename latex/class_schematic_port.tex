\doxysubsection{Detailed Description}
\begin{DoxyVerb}  \brief The class provides method to draw and andle a SchematicPort.

  A SchematicPort is a child af a SchematicNode and is used to link a
\end{DoxyVerb}
 parent node to other nodes. It can be an input port, or better , a port used to accept link coming from other node. A port that isn\textquotesingle{}t an input port cannot accept links but can begin to draw links.~\newline
A \mbox{\hyperlink{class_schematic_port}{Schematic\+Port}} has got a hook thet is a position where links starts or ends.~\newline
 A \mbox{\hyperlink{class_schematic_port}{Schematic\+Port}} can be linked to an arbitrary number of links. A \mbox{\hyperlink{class_schematic_port}{Schematic\+Port}} handles a container of all links to retrieve all linked node; each link is indexed using a progressive number assigned when the link is inserted to the container. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_schematic_node}{Schematic\+Node}}, \mbox{\hyperlink{class_schematic_link}{Schematic\+Link}}. 
\end{DoxySeeAlso}
