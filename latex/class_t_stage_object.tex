\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_t_stage_object}{TStage\+Object}} is the Toonz class representing animatable scene objects like columns, pegbars, tables and cameras (pretty much every object in Toonz\textquotesingle{}s stage schematic).

\begin{DoxyParagraph}{Description}

\end{DoxyParagraph}
A \mbox{\hyperlink{class_t_stage_object}{TStage\+Object}} contains a common set of editable parameters (coordinates, angles and more) that define the object\textquotesingle{}s state along an xsheet timeline. These parameters are in the form of animatable \mbox{\hyperlink{class_t_double_param}{TDouble\+Param}} instances and can be accessed directly through the \mbox{\hyperlink{class_t_stage_object_ae4cf64f5085310a14074ceb54d2b5177}{get\+Param()}} method. ~\newline
~\newline
 A notable responsibility of the \mbox{\hyperlink{class_t_stage_object}{TStage\+Object}} class is that of building the keyframes data that will be shown in xsheet viewes (the key icons). The object\textquotesingle{}s keyframes are built and cached each time one of its associated parameters is changed. ~\newline
~\newline
 Furthermore, a \mbox{\hyperlink{class_t_stage_object}{TStage\+Object}} instance is typically a node in a tree-\/like hierarchycal structure of class \mbox{\hyperlink{class_t_stage_object_tree}{TStage\+Object\+Tree}} which is used to define relative object placement between objects.

\end{DoxyParagraph}
