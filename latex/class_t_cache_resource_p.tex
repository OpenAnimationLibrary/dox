\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_cache_resource_p}{TCache\+ResourceP}} class implements \textquotesingle{}smart\textquotesingle{} referenced pointers to \mbox{\hyperlink{class_t_cache_resource}{TCache\+Resource}} instances.

Smart pointers are typically used to hold references to data that is automatically freed once it is no more referenced by any smart pointer instance. The \mbox{\hyperlink{class_t_cache_resource_p}{TCache\+ResourceP}} is a specialized smart pointer-\/like class that references \mbox{\hyperlink{class_t_cache_resource}{TCache\+Resource}} instances. ~\newline
 ~\newline
In order to retrieve a Cache resource, it is sufficient to know the resource name and invoke the {\ttfamily \mbox{\hyperlink{class_t_cache_resource_p}{TCache\+Resource\+P(const std\+::string\&, bool)}}} constructor. The bool optionally specified as second argument specifies whether the resource has to be created if none is found with passed name.
