\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_xsh_child_level}{TXsh\+Child\+Level}} class provides a child level of xsheet and allows its management.

Inherits {\bfseries{\mbox{\hyperlink{class_t_xsh_level}{TXsh\+Level}}}}. ~\newline
 A child level \mbox{\hyperlink{class_t_xsh_child_level_a939e4baa485b3bdb71cec0f7597b17ea}{get\+Child\+Level()}} is a xsheet reduced to level. The class allows to edit xsheet \mbox{\hyperlink{class_t_xsh_child_level_ad2676276ae1db83a898a3eef10e0ace8}{get\+Xsheet()}}, change xsheet \mbox{\hyperlink{class_t_xsh_child_level_af6dba0cad3385e973a8048de8a3da8d3}{set\+Xsheet()}} and change scene xsheet \mbox{\hyperlink{class_t_xsh_child_level_a10691f4c9edb22c1b8af2c1c2bab4f21}{set\+Scene()}}. It\textquotesingle{}s possibile change level name set\+Name() and know how many frame are in level \mbox{\hyperlink{class_t_xsh_child_level_a4ac82b11f381c81901cf5736bdf231f1}{get\+Frame\+Count()}}.

The class provides methods to control child level icon, make\+Icon(), get\+Icon() and invalidate\+Icon().
