\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_style_sample}{Style\+Sample}} class provides to view a square color. 

Inherits {\bfseries{QWidget}}.

By default square color is set to {\bfseries{TPixel32(235,235,235,255)}}, you can set other color using \mbox{\hyperlink{class_d_v_gui_1_1_style_sample_adc80e6e4ca3bf07834b1ff06ca78cb25}{set\+Color()}}; you can also define current color with a style {\bfseries{\mbox{\hyperlink{class_t_color_style}{TColor\+Style}}}}, {\bfseries{\mbox{\hyperlink{class_d_v_gui_1_1_style_sample_a76caeb2e0f26209e6056c1fc46b7430d}{get\+Style()}}}}, using \mbox{\hyperlink{class_d_v_gui_1_1_style_sample_a5564a01698c526ec6bbddeaa1035c961}{set\+Style()}}. You can pass to constructor square size.

\mbox{\hyperlink{class_d_v_gui_1_1_style_sample}{Style\+Sample}} permit to manage click event, it\textquotesingle{}s possile to enable this feature setting \mbox{\hyperlink{class_d_v_gui_1_1_style_sample_a4811227f8876ac58cc9ad2d68b8049a2}{enable\+Click(bool on)}} to true. If it is enable when click in square class emit the signal clicked(const TColor\+Style \&style).
