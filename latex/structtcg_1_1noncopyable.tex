struct tcg\+::noncopyable$<$ B $>$}
The noncopyable class can be inherited to forbid access to the copy constructor and assignment operator. 

\begin{DoxyNote}{Note}
A template parameter is provided to permit the empty base class optimization. In case this optimization is not needed, please use boost\+::noncopyable instead. 
\end{DoxyNote}
