\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_free_layout}{Free\+Layout}} class implements a layout whose items can be freely positioned, ignoring parent size bounds or geometry. 

A \mbox{\hyperlink{class_free_layout}{Free\+Layout}} instance can be used to achieve results similar to that of a QScroll\+Area, with a more straightforward API and without restriction to just one item at a time. Please observe that layouts can be inserted too. \begin{DoxyVerb}      Specifically, FreeLayout is a convenience subclass of DummyLayout
\end{DoxyVerb}
 whose items\textquotesingle{} sizes are enforced to their size\+Hint() -\/ in order to spare users from having to deal with it externally. This complies with the idea that its items are comfortly free to adapt to their preferred size, as it would be in a scrollable environment.
