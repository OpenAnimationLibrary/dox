The \mbox{\hyperlink{class_style_editor_g_u_i_1_1_plain_color_page}{Plain\+Color\+Page}} is used to control the color parameter. 

\begin{DoxyVerb}        Inherits \b StyleEditorPage.

        The PlainColorPage is made of a \b SquaredColorWheel and a \b
\end{DoxyVerb}
 \mbox{\hyperlink{class_style_editor_g_u_i_1_1_color_slider}{Color\+Slider}}, a collection of {\bfseries{\mbox{\hyperlink{class_style_editor_g_u_i_1_1_color_channel_control}{Color\+Channel\+Control}}}}, and a number of radio button (to control the Color\+Wheel behaviour).
