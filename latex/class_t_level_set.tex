\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_t_level_set}{TLevel\+Set}} is a typical \mbox{\hyperlink{class_toonz_scene}{Toonz\+Scene}} component that is used to trace all levels in a Toonz scene.

The main purpose of the \mbox{\hyperlink{class_t_level_set}{TLevel\+Set}} is that of acting as a container of {\bfseries{ externally owned \texorpdfstring{$<$}{<}\textbackslash{}B\texorpdfstring{$>$}{>} TXsh\+Level$\ast$ instances. The container is internally implemented as a vector, and shares its access model. ~\newline
~\newline
Additionally, the set is used to map the stored levels by their names (case-\/insensitive comparisons), and associates a {\itshape folder} to each of the level instances. The level folder is not physical (ie it does not refer to a directory in the file system), but can be used to list the levels pool in a hierarchycal view.}}

{\bfseries{\begin{DoxyWarning}{Warning}
This class distinguishes between {\bfseries{paths}} and {\bfseries{folders}} as described above, but the associated data type (\mbox{\hyperlink{class_t_file_path}{TFile\+Path}}) is the same. Please, pay attention to the function argument names, in this respect. 
\end{DoxyWarning}
