\doxysubsection{Detailed Description}
The class \mbox{\hyperlink{class_t_stage_object_id}{TStage\+Object\+Id}} permits the specialization of a generic stage object.

The possible type of a stage object are\+: table Table\+Id, camera Camera\+Id, pegbar Pegbar\+Id, column Column\+Id and spline Spline\+Id; the class allows to know object type\+: \mbox{\hyperlink{class_t_stage_object_id_ab597e1e86fc3b5e64d552bec77f86b4d}{is\+Camera()}}, \mbox{\hyperlink{class_t_stage_object_id_a2ab00aa1e5039ac19b1bf7ab0dead466}{is\+Table()}}, \mbox{\hyperlink{class_t_stage_object_id_a87a62052c05b8bfb6cb28c05f555a718}{is\+Pegbar()}}, \mbox{\hyperlink{class_t_stage_object_id_af6d13fcf31e7e10eef28f085855f469c}{is\+Column()}} and \mbox{\hyperlink{class_t_stage_object_id_af23c7c57b551d078a40ff429942d501e}{is\+Spline()}}. ~\newline
 For each specialization you give to each stage object a sequential number. The class provides methods to get sequential number get\+Index() and object\+Id name \mbox{\hyperlink{class_t_stage_object_id_a2a13bb363b1662a7c5e70fdafe9bed5f}{to\+String()}}. It\textquotesingle{}s possible to compare different stage object using defined operator, \mbox{\hyperlink{class_t_stage_object_id_a41aa3137a5a0105215e5445f7eb2eb81}{operator==()}}, \mbox{\hyperlink{class_t_stage_object_id_a6fdcc6e9f867d75da8e7d17b12d16a14}{operator!=()}}, \mbox{\hyperlink{class_t_stage_object_id_a9a554d06a33cd89a6c689fd1e9653ef7}{operator$<$()}}, \mbox{\hyperlink{class_t_stage_object_id_a1e66014b32ee7978d7989c8920c4397a}{operator$<$=()}}, \mbox{\hyperlink{class_t_stage_object_id_a2db9c77b22386a980dc9df5f931f048a}{operator$>$()}} and \mbox{\hyperlink{class_t_stage_object_id_aba8c69e1d9b5f89a72c2b4ada8a2c301}{operator$>$=()}}.
