\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_popup_button}{Popup\+Button}} represents an enum type in a compact, popup-\/based form. A \mbox{\hyperlink{class_popup_button}{Popup\+Button}} instance at rest is a simple push button showing an icon which represents the stored enum. As the user clicks on the button, a popup appears showing a description and, optionally, an icon of the selectable enum values.
