\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Pix, typename Val$>$\newline
class TRop\+::borders\+::pixel\+\_\+selector$<$ Pix, Val $>$}
This class is just a stub for pixel selector classes.

Pixel Selectors are objects used in raster borders recognition to interpret a raster\textquotesingle{}s pixels and direct edge iterators. ~\newline
~\newline
 The main purpose of this class is that of distinguishing pixels from their {\itshape  values \texorpdfstring{$<$}{<}\textbackslash{}I\texorpdfstring{$>$}{>}. Values are the actual entities recognized in the borders extraction process -\/ adjacent pixels that are cast to the same value will be included in the same border. ~\newline
~\newline
 For example, if we want to extract the borders of a fullcolor image that separate bright areas from dark ones, we can just have a pixel selector cast each pixel to the integer value 0 if the pixel\textquotesingle{}s value is below the brightness threshold, and 1 if above. ~\newline
~\newline
 Also, pixel selectors tell whether pixels are to be intended as completely transparent (virtual pixels outside the raster boundaries are). }
