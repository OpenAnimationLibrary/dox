\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Node\+Content\+Type, typename Arc\+Type$>$\newline
class Graph$<$ Node\+Content\+Type, Arc\+Type $>$}
\mbox{\hyperlink{class_graph}{Graph}} class used by the centerline vectorization process. 

Introducing a directed graph structure that allows local access\+: main feature is that a graph edge physically belongs to the node that emitted it, by storing it in a \textquotesingle{}link vector\textquotesingle{} inside the node. No full-\/scale edge search method is therefore needed to find node neighbours. \begin{DoxyVerb}      No specific iterator class is needed, just use unsigned ints to
\end{DoxyVerb}
 perform random access to nodes and links vectors.
