\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field}{Int\+Pair\+Field}} class provides to view an object to manage a pair of int parameter. 

Inherits {\bfseries{QWidget}}.

The object is composed of an horizontal layout {\bfseries{QHBox\+Layout}} which contains two pair \mbox{[}label, text field\mbox{]} and a slider with two grab, one for each int value to manage. \mbox{[}label, text field\mbox{]} are a {\bfseries{QLabel}} and a {\bfseries{\mbox{\hyperlink{class_d_v_gui_1_1_int_line_edit}{Int\+Line\+Edit}}}}. Objects are inserted in the following order\+: a label and respective text field, the slider and the other pair \mbox{[}lebel,text field\mbox{]}. Object size width is not fixed, while height is equal to DVGui\+::\+Widget\+Height, 20; labels width depend from its text length, text fields has fixed size, while slider width change in according with widget size.

Objects contained in this widget are connected, so if you change one value the other automatically change, if it is necessary. You can set current value, \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a98ec91188e1c1507e945918c14e5ffeb}{get\+Values()}}, minimum and max value, \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a0fb6fb489a5f962c23c20a6549447f9c}{get\+Range()}}, using \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a07e283f0fb7adf6f8b49a2aacd83a64b}{set\+Values()}}, \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a02f7dfac0d4ecf40bd4b848031e2ce19}{set\+Range()}}. \begin{DoxyVerb}         To know when one of two int parameter value change class
\end{DoxyVerb}
 provides a signal, \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a84c380fac32838d260440760c3622f18}{values\+Changed()}}; class emit signal when a grab slider position change or when editing text field, left or right is finished and current value is changed. Editing text field finished may occur if focus is lost or enter key is pressed. See SLOT\+: \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a6a2479d283ad86f3498c1f380eb11b95}{on\+Left\+Editing\+Finished()}} and \mbox{\hyperlink{class_d_v_gui_1_1_int_pair_field_a2dd84a1e70f9728ef12fc3c597ba5fcd}{on\+Right\+Editing\+Finished()}}.
