\doxysubsection{Detailed Description}
Data class containing the relevant additional information needed in a render process. 

This class stores most of the data required by Toonz and single fxs to perform a render. While a good part of this data group is used internally by Toonz to deal with global aspects of a rendering process, another part must be actively referenced when rendering an fx.

The most important member of this class is {\ttfamily m\+\_\+affine}, which specifies the affine transform that must be applied by an fx if it were to render on the default reference.

TRaster\+Fx\+::compute() for further information. 
\end{DoxySeeAlso}
