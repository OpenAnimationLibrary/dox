\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_style_editor_g_u_i_1_1_settings_page}{Settings\+Page}} is used to show/edit style parameters. 

This class stores the GUI for editing a {\itshape copy} of the current color style. Updates of the actual current color style are {\itshape not} performed directly by this class.
