generic tree model. It maintains internally a tree structure of \mbox{\hyperlink{class_tree_model_1_1_item}{Tree\+Model\+::\+Item}}\textquotesingle{}s. Each QModel\+Index created by the \mbox{\hyperlink{class_tree_model}{Tree\+Model}} points internally to the corrispondent \mbox{\hyperlink{class_tree_model_1_1_item}{Tree\+Model\+::\+Item}}
