\doxysubsection{Detailed Description}
Provide a structure representing a node af a tree. This class maintains information about a node of the stage schematic and its children. The tree is generated using the \mbox{\hyperlink{class_t_stage_object_tree}{TStage\+Object\+Tree}}. The generated tree is used to oreder stage schematic nodes in the window.
