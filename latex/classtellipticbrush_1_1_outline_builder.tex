\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classtellipticbrush_1_1_outline_builder}{Outline\+Builder}} class is the object used to translate centerline points into outline points. The purpose of this class is to take a \mbox{\hyperlink{structtellipticbrush_1_1_centerline_point}{Centerline\+Point}} instance and build a couple of outline points -\/ at either side of the centerline -\/ eventually adding (cap) points to form a smooth outline.
